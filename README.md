# Exercise to handle convnets

Guided exercise to
generate images
define a convolution network
train it on a classification task on synthetic data
## Workspace setup
Preferably use in a python virtual environment (or a conda environment)
## Create environment
```
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```
## Using the environment
```
source venv/bin/activate
```
```
deactivate // to use system Python
```
## Update the environment
```
pip install <additional-package>
pip freeze > requirements.txt
```
